\FACTFinder\Core\Server

\FACTFinder\Core\ServerMultiCurlDataProvider

This implementation retrieves the FACT-Finder data by using the cURL's "multi interface".

Responses are queried in parallel and lazily and are cached as long as parameters don't change.
