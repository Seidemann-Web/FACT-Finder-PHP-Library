\FACTFinder\DataFilterStyle

Enum for style of filter groups within the After Search Navigation (ASN).

Of course, this is just a workaround for enums in PHP. It's a bit convoluted but in terms of usage, it seems to be one of the cleanest solutions. There are no further doc blocks, so please see the source code for further explanations.
