\FACTFinder\Core\Server

\FACTFinder\Core\ServerFileSystemDataProvider

This implementation retrieves the FACT-Finder data from the file system. File names are generated from request parameters. For the naming convention see the getFileName() method.

Responses are queried sequentially and lazily and are cached as long as parameters don't change.
