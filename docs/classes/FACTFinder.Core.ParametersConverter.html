removeIgnoredParameters()

removeIgnoredParameters(\FACTFinder\Core\Parameters  $parameters, array<mixed,boolean>  $ignoreRules)

Removes keys from a Parameters object according to the given ignore rules. It basically turns the parameters into the set difference of the parameters and the ignore rules based on keys.

Parameters

Array of parameters to be ignored. The keys are the parameter names, the values are simply "true", but could technically have any value.

applyWhitelist()

applyWhitelist(\FACTFinder\Core\type  $parameters, array<mixed,boolean>  $whitelistRules)

Removes keys from a Parameters object according to the given whitelist rules.

An empty whitelist means do NOT apply any whitelist (anything is allowed). It removes any keys that are not keys in the given whitelist array aswell. If the key of any rule starts with '/' the key is interpreted as a regular expression to be matched against.

Parameters

\FACTFinder\Core\type $parameters
array<mixed,boolean> $whitelistRules

Array of parameters to be allowed. The keys are the parameter names, the values are simply "true", but could technically have any value.
